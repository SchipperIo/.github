## What We're Building

### ⚡ IonStream

> A high-performance distributed streaming platform — pure .NET, no JVM, no ZooKeeper.

IonStream is a self-contained streaming queue built on Raft consensus and gRPC transport. A full cluster is a single binary. It competes directly with Kafka, Redpanda, and NATS JetStream — with a .NET-native API, native AOT compilation, and first-class clients for .NET, TypeScript, Python, Go, Rust, and Java.

**Key capabilities:**
- Raft consensus with ISR replication, follower reads, and dynamic node addition
- Append-only memory-mapped log with compaction and retention
- Built-in retry queues and Dead Letter Queue (DLQ) with a management dashboard
- Cross-cluster active/active replication
- OpenTelemetry metrics, included Grafana dashboards, and a live web dashboard
- Native AOT binaries for `linux-x64` and `win-x64`
